Creating User Documentation

Class Information

This class is about learning how to create documentation for a software we choose near the start of the class. We are learning how to make tutorials and instruction manuals.

Major Assignments

  • User Survey
  • Usability Testing
  • E-Portfolio

My Thoughts on the Class

This class is similar, yet distinctly different from Developing Intelligent Content, which I find makes it interesting to be taking them simultaneously. I have been enjoying the class projects thus far, especially because I chose Steam, a PC gaming platform, as my software. I am very familiar with using it, which makes instructional assignments easy, and the survey went super quick because I know so many people who use it.
